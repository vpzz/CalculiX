# English
1. Clone this repository.
2. Change directory to CCX. Enter `make -j` command, if everything goes well, you will get an executable file `ccx_2.22`.
3. Unzip the file `ccx_2.22.test.tar.bz2` to the `CCX/CalculiX/CCX_2.22/test` directory, cd to this directory, enter `./compare` to perform test. Observe whether an `error.pid` file is generated.
# 中文
1. 克隆此存储库。
2. 进入CCX目录。输入`make -j`命令，如果一切顺利，您将得到一个可执行文件 `ccx_2.22`。
3. 解压ccx_2.22.test.tar.bz2到CCX/CalculiX/CCX_2.22/test目录中，进入该目录，输入 `./compare` 命令，以执行测试。观察是否会产生error.pid文件
